# 使用

```groovy
implementation 'com.xuie:util-tools:1.2.1'
```


# 部署依赖至服务器

- 在`gradle.progerties`中添加user-key
```
bintray.user=xuie0000
bintray.apikey=<your api key>
```

- `./gradlew install`
- `./gradlew bintrayUpload`

> **Windows10用命令执行，窗口点有BUG**，其他系统还没有试
